Subjects will include:
•        Modeling and representation
2D/3D transformations, 2D/3D geometry, forward and inverse kinematics, motion representations, configuration space, Lagrangian dynamics.
•        Control and planning
Feedback control and asymptotic stability, optimal and model predictive control, motion planning, task planning
•        Perception
Uncertainty models, Kalman filters, visual sensors, 3D mapping, calibration, some computer vision.
•        Software and hardware system integration
Simulation software, visualization and GUIs, distributed system middleware (ROS), performance evaluation

This repo is for demonstration purpose only, and any other usage is prohibited.
